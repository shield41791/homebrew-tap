# Shield41791 Tap

## How do I install these formulae?

`brew install shield41791/tap/<formula>`

Or `brew tap shield41791/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "shield41791/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
